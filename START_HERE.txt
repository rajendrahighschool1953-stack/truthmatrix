## âš¡ QUICK START (Choose Your Platform)

### FOR WINDOWS USERS:
1. Double-click `setup.bat` in the project folder
2. When asked, enter your Gemini API key
3. Wait for installation (takes 2-3 minutes)
4. When done, it will tell you to run: npm run dev
5. Open http://localhost:3000 in your browser

### FOR MAC/LINUX USERS:
1. Open Terminal in the project folder
2. Run: chmod +x setup.sh
3. Run: ./setup.sh
4. When asked, enter your Gemini API key
5. When done, run: npm run dev
6. Open http://localhost:3000 in your browser

### MANUAL SETUP (All Platforms):
```bash

# Step 3: Install dependencies
npm install

# Step 4: Start development server
npm run dev

# Step 5: Visit http://localhost:3000
```

## â“ TROUBLESHOOTING

**Q: "GEMINI_API_KEY not configured"**
A: Create .env.local file with your API key. Restart server.

**Q: "Port 3000 already in use"**
A: Run: npm run dev -- -p 3001

**Q: "npm install fails"**
A: Delete node_modules, run npm cache clean --force, then npm install

**Q: "Image upload fails"**
A: Make sure file is < 10MB and is PNG/JPG/GIF format

See **SETUP_GUIDE.md** for more help.
